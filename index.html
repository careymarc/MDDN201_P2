[WEB-CHECK]

WEB-CHECK is a free to use application that people can use to check if a certain website is safe. This application can be used on mobile devices and can be a downloadable extension for your web browser. There will also be a website for users to go to if they don’t want to download the extension/application. The purpose of WEB-CHECK is to check how safe a website is. People can copy and paste a url of a website and input it in the bar. When the user presses the “enter” button, a gauge will below the bar. When the enter button has been pressed the gauge will indicate how safe the website is. The user will be able to tell by the gauge as it will have three colours, red, yellow and green. Green will mean that the website is safe and the user doesn’t have to worry about a thing. While yellow means that the website is not 100% safe, but the website is not the worst website out there. If the gauge is pointing towards the red section then that means that the website is not safe for the user. This website could potentially take the users data when they are browsing that website. Or they could infect their computer with a virus or other harmful stuff that could damage their computer. WEB-CHECK will be able to tell the user if the website could be doing this. Below the gauge will be some information into why the website is not safe. So depending on the website it could say that the users that interact with the website, their data gets taken and can used by other companies/people that will fine this data useful. Also this will make the user more aware when using websites from now on. Hopefully this product will be able to make people’s interaction with the internet more friendly and safe. A company known as Cambridge Analytica harvested millions of people’s data on the social media platform FaceBook. In 2014 that built a system to target US voters and could predict their outcome. Also making advertisements personalised for every person. By doing this it affected the choices that people made and impacted on the election. FaceBook has also been selling their users data to third parties. Also on the internet there are hackers that may steal your information, such as credit card information or other personal information. WEB-CHECK will be able to check if that the websites your on could do any of these things. For the design aspect of it using primary colour will allow the user to identify the “good”, “average” and “bad” on the gauge. The colours for the gauge will be green, yellow and red. For the other parts of the gauge will have white, black and two different types of grey for the pointer/indicator. The background will be grey as the colours will not class with the primary and the indicator on the gauge. The input bar will be in white and next to the bar will be a blue button for when the user wants to check if they website is safe or not. The header font will be Noto Serif and the text font will be in Noto Sans JP. The icon will the gauge with a little rod connected to it. This will make it look like a magnifying glass. The downloadable application will also have this icon and it will have a blue background behind it.
